Main Points from Stage 0: Getting Started with HTML

How the Web Works

The Web is a collection of HTML documents. The language of these documents is HTML (Hyper Text Markup Language). Software used to interpret these HTML documents into a format that is pleasing to users of the Web is called browsers.

HTTP (Hyper Text Transfer Protocol) is the protocol used by browsers to request information from servers, computers where HTML and associated files reside.

HTML

HTML is made up of:

-text content (what you see)
-markup (what it looks like)
-references to other documents (like images or videos)
-links to other pages

HTML Markup

HTML is composed of elements. These elements are further characterized by tags. Tags assign characteristics or attributes to elements. For example, there are tags that instruct brosers to display text in bold and italics. These tags can be combined and "layered" resulting in text that is both bold and italicized.

The bold tag in HTML looks like this:
[b] contents [/b]
where the brackets are replaced by the less than and greater than signs.

The emphasis tag, aka italicize tag, looks like this:
[em]contents[/em]
again, where the bracets are replaced by the less than and greater than signs.

HTML Attributes

Some HTML elements contain attributes or language that has a specific function like creating a hyperlink or displaying an image from a file. In general, the format of an HTML element that has an attribute looks like this:
[Tag Attr = "value"]contents[/tag]
where Tag is the tag name, Attr is the attribute name and value is substituted with a web address or file location among other values.

The anchor tag creates a hyperlink and the general format looks like this:
[a href = "www.udacity.com"]practice link[/a]

The img tag inserts an image from a file location. In general the format looks like this:
[img src = "url" alt="text"]
where img is the tag name, src is the attribute name, and url is the file location. The last part of the element is used by the browser to display text in case the URL is broken and no image is shown. The text that is displayed can also be used to describe the image and can be of benefit to the visually challenged.

Inline and Block elements

HTML elements are either inline or block. If they are block, then they have an "invisible block" around them. This is important for formating purposes and will be developed in future stages of this course.
